    SSAI: Detección de bloques de anuncios y conmutación por error automática

    En este tema, aprenderá a activar la configuración de conmutación por error automática de Brightcove Player para solicitar anuncios SSAi (Inserción de anuncios del lado del servidor) cuando se detecten bloqueadores de anuncios.

    Resumen

    Con la conmutación por error de anuncios activada, cuando Brightcove Player detecte un bloqueador de anuncios presente en el navegador, el reproductor solicitará automáticamente una transmisión SSAI. De lo contrario, utilizará anuncios de IMA.

    La conmutación por error de anuncios SSAI requiere lo siguiente:

    • Complemento IMA3 v3.2.0+
    • Complemento SSAI v1.3.1+
    • Brightcove Player versión 6.8.0+

    Advertencias

    • La detección de bloqueador de anuncios es muy confiable, pero no al 100%.
    • Los clientes que utilicen esta función deben utilizar Entrega dinámica.
    • Solo se admiten complementos IMA y SSAI (sin FreeWheel).

    Introducción

    Para reproducir anuncios del servidor desde Video Cloud, debe seguir estos pasos:

    1. Crear una configuración de anuncio
    2. Crear un reproductor de Brightcove
    3. Implementar la conmutación por error automática mediante Studio

    Crear una configuración de anuncio

    La configuración del anuncio define varios aspectos de la reproducción de SSAI, incluidas las llamadas a anuncios, los indicadores y otras opciones de configuración. La respuesta al anuncio puede ser reglas de anuncios VAST, VMAP o DFP. Para crear una configuración de anuncio, siga estos pasos:

    1. En Video Cloud Studio, expanda el menú ADMIN y seleccione Configuración de anuncios del lado del servidor.

    2. Añade información para la configuración de tu anuncio y selecciona Guardar.

    Para obtener más información, consulte el documento Configuración de la configuración de anuncios del lado del servidor .

    Para crear una configuración de anuncio mediante la API SSAI, consulte el documento de la API de configuración de anuncios SSAI de Video Cloud.

    Crear un reproductor de Brightcove

    Cree un nuevo reproductor de Brightcove con Video Cloud Studio. Para obtener más información, consulte Inicio rápido: Creación y estilo de un documento del Reproductor .

    Debe utilizar una versión 6.8.0+ de Brightcove Player.

    Implementación de conmutación por error automática mediante Studio

    La forma más sencilla de configurar el reproductor para anuncios de conmutación por error automática es con Video Cloud Studio. Una vez que haya creado una configuración y un reproductor de anuncios, estará listo para configurar el reproductor para la conmutación por error automática de la siguiente manera:

    1. Abra el módulo PLAYERS y localice el reproductor al que desea añadir la funcionalidad de publicidad.
    2. Seleccione el enlace para que el reproductor abra las propiedades del reproductor.
    3. Seleccione Publicidad en el menú de navegación de la izquierda.
    4. Marque la casilla de verificación Habilitar cliente (IMA) .

    5. Incluya la URL de su etiqueta de anuncio IMA. Para este ejemplo, utilizaremos la URL de etiqueta de anuncio de muestra.

      Para obtener más información sobre las propiedades de publicidad del reproductor, consulte el documento Configuración de la publicidad del reproductor mediante el módulo de reproductores .

    6. Marque la casilla de verificación Habilitar servidor (SSAI) .

      La opción del lado del servidor no aparecerá a menos que su cuenta haya sido habilitada para SSAI.
    7. En el menú desplegable Seleccionar configuración , seleccione la configuración del anuncio que desea asociar a este reproductor.
    8. Si desea que las superposiciones se muestren sobre sus anuncios, active la casilla de verificación Habilitar superposiciones de información de anuncios. Esto incluye las superposiciones «Más información» y la cuenta regresiva de anuncios.

    9. La detección de bloques de anuncios con conmutación por error de anuncios del lado del servidor se habilita automáticamente cuando se habilita tanto los anuncios del lado del cliente como del servidor.

    10. Seleccione Guardar.

    11. Para publicar el reproductor, seleccione Publicar e incrustar > Publicar cambios.

      Eso es todo lo que necesita hacer para habilitar la conmutación por error automática para su reproductor.

    Reproducir un vídeo con anuncios

    Cualquier vídeo que recupere de Video Cloud que se haya ingerido con Entrega dinámica incluirá los anuncios IMA del lado del cliente cuando no se detecte ningún bloqueador de anuncios. Cuando se detecte un bloqueador de anuncios, el vídeo incluirá anuncios del lado del servidor especificados en el archivo VMAP en la configuración del anuncio. Tenga en cuenta que el vídeo necesita tener una pista de audio asociada a él para que SSAI funcione.
