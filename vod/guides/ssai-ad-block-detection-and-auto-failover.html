Descripción general

Con la conmutación por error de anuncios activada, cuando Brightcove Player detecta un bloqueador de anuncios presente en el navegador, el reproductor solicitará automáticamente una transmisión SSAI. De lo contrario, utilizará anuncios IMA.

La conmutación por error de anuncios SSAI requiere lo siguiente:

  • Complemento IMA3 v3.2.0 +
  • Complemento SSAI v1.3.1 +
  • Versión de Brightcove Player 6.8.0+

Advertencias

  • La detección de bloqueadores de anuncios es muy confiable, pero no al 100%.
  • Los clientes que utilicen esta función deben utilizar Dynamic Delivery.
  • Solo se admiten los complementos IMA y SSAI (no FreeWheel).

Empezando

Para reproducir anuncios del lado del servidor desde Video Cloud, debe seguir estos pasos:

  1. Crea una configuración de anuncios
  2. Crea un reproductor Brightcove
  3. Implementar la conmutación por error automática con Studio

Crea una configuración de anuncios

La configuración de anuncios define varios aspectos de la reproducción SSAI, incluidas las llamadas de anuncios, balizas y otras opciones de configuración. La respuesta de su anuncio puede ser VAST, VMAP o DFP Ad Rules. Para crear una configuración de anuncios, siga estos pasos:

  1. En Video Cloud Studio, expanda el ADMINISTRACIÓN menú y seleccione Configuración de anuncios del lado del servidor.

  2. Agregue información para la configuración de su anuncio y seleccione Ahorrar.

Para obtener más detalles, consulte la Configuración de la configuración de anuncios del lado del servidor documento.

Para crear una configuración de anuncios con la API SSAI, consulte la API de configuración de anuncios SSAI de Video Cloud documento.

Crea un reproductor Brightcove

Cree un nuevo reproductor Brightcove con Video Cloud Studio. Para obtener más detalles, consulte la Inicio rápido: Creación y estilo de un reproductor documento.

Debe utilizar Brightcove Player versión 6.8.0+.

Implementación de la conmutación por error automática con Studio

La forma más sencilla de configurar su reproductor para anuncios de conmutación por error automática es con Video Cloud Studio. Una vez que haya creado una configuración de anuncios y un reproductor, estará listo para configurar el reproductor para la conmutación por error automática de la siguiente manera:

  1. Abre el JUGADORES módulo y ubique el reproductor al que desea agregar funcionalidad publicitaria.
  2. Seleccione el enlace del jugador para abrir las propiedades del jugador.
  3. Seleccione Publicidad en el menú de navegación de la izquierda.
  4. Comprobar el Habilitar el lado del cliente (IMA) caja.

  5. Incluya la URL de su etiqueta publicitaria de IMA. Para este ejemplo, usaremos la URL de etiqueta publicitaria de muestra.

    Para obtener detalles sobre las propiedades publicitarias del reproductor, consulte la Configurar la publicidad de los jugadores mediante el módulo de jugadores documento.

  6. Comprobar el Habilitar el lado del servidor (SSAI) caja.

    La opción del lado del servidor no aparecerá a menos que su cuenta haya sido habilitada para SSAI.
  7. Desde el Seleccionar configuración menú desplegable, seleccione la configuración de anuncios que le gustaría asociar con este reproductor.
  8. Si desea que se muestren superposiciones sobre sus anuncios, marque la casilla Habilitar superposiciones de información de anuncios caja. Esto incluye "Más información" y superposiciones de cuenta regresiva de anuncios.

  9. La detección de bloqueos de anuncios con conmutación por error de anuncios del lado del servidor se habilita automáticamente cuando habilita los anuncios del lado del cliente y del lado del servidor.

  10. Seleccione Ahorrar.

  11. Para publicar el reproductor, seleccione Publicar e insertar> Publicar cambios.

    Eso es todo lo que necesita hacer para habilitar la conmutación por error automática para su reproductor.

Reproducir un video con anuncios

Cualquier video que recupere de Video Cloud que se haya ingerido con Dynamic Delivery incluirá los anuncios IMA del lado del cliente cuando no se detecte ningún bloqueador de anuncios. Cuando se detecta un bloqueador de anuncios, su video incluirá anuncios del lado del servidor especificados en el archivo VMAP en su configuración de anuncios. Tenga en cuenta que el video debe tener una pista de audio asociada para que SSAI funcione.
