Descripción general

Con SSAI, los anuncios se unen a su flujo de contenido en el lado del servidor. La señalización de eventos de error de anuncios VAST se proporciona cuando se usa VOD + SSAI.

Los eventos de error de anuncios brindan información sobre los flujos de trabajo de sus anuncios, lo que le permite resolver problemas de manera proactiva. Asegurarse de que sus anuncios se publiquen ayudará a maximizar los ingresos publicitarios.

Balizas de error de anuncios

La siguiente tabla define los eventos de error de anuncios transmitidos a través de SSAI, como se define en las especificaciones de VAST 3.0 / 4.0.

Código de error Tipo Descripción
100 ErrorCodeXMLParse

Error de análisis XML

  • Error en la transformación XSLT
  • La respuesta estaba vacía
200 ErrorCodeTrafficking

Error de tráfico. El reproductor de video recibió un tipo de anuncio que no esperaba y / o no puede mostrar.

  • Se devolvió una respuesta de anuncio válida, pero no se encontraron anuncios lineales
  • Se devolvió un formato de anuncio no progresivo.
    • VPAID swf (cuando no se espera como transferencia)
    • Tipo de mimo de video no válido
301 ErrorCodeWrapperRequest

Tiempo de espera de VAST URI proporcionado en el elemento Wrapper, o de VAST URI proporcionado en un elemento Wrapper posterior. Incluye errores de solicitud como URI no válido, inaccesible o tiempo de espera de solicitud para URI y seguridad u otras excepciones relacionadas con la solicitud de un URI de VAST.

  • URI de contenedor no disponible: 4xx, 5xx
  • El contenedor no respondió dentro del umbral de tiempo de espera establecido
302 ErrorCodeWrapperLimit

Se alcanzó el límite de envoltura, según lo definido por el reproductor de video. Se han recibido demasiadas respuestas de Wrapper sin respuesta InLine.

  • Superó el límite de seguimiento de la envoltura
303 ErrorCodeWrapperNoAds

No hay respuesta VAST de anuncios después de uno o más Wrappers. También incluye el número de respuestas VAST vacías de reserva.

  • No se recibió una respuesta de anuncio lineal (progresiva) a través de solicitudes de envoltura
  • La respuesta del contenedor estaba vacía
406 ErrorCodeMissingMezz

Se requirió entrepiso pero no se proporcionó. Anuncio no servido.

  • Implementaciones estrictas de VAST 4.0 y exigir que se devuelvan anuncios Mezz. Podría ser viable para los dispositivos OTT garantizar la calidad.
  • VAST 4 específico
407 ErrorCodeProcessingMezz

Mezzanine se descargó por primera vez. Anuncio no servido.

  • Siempre que se reciba una creatividad válida, pero se elimine debido al procesamiento, este evento debe activarse.
  • VAST 4 específico
900 ErrorCodeUndefined

Error indefinido

  • Ocurrió un error generalizado y no está bien definido por otros errores
